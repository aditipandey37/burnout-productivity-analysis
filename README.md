# burnout-productivity-analysis
An analytical study of employee burnout and its impact on productivity, integrating Python-based data processing with Power BI dashboards for insights.

## Objective
- Analyze burnout-related factors
- Understand their impact on productivity
- Present insights using interactive dashboards

## Tools & Technologies
- Python (Pandas, NumPy)
- Power BI
- Jupyter Notebook
- CSV / Excel datasets

## Dataset Source
Kaggle (public dataset used for educational analysis)
